⚡ Quick Start

Get up and running in 3 steps:

  • Install dependencies: npm install
  • Start collaboration server: npm run collab-server
  • Import and use: See code below
import { useCollaboration, CollaborativeEditor } from './collaboration'; function MyWorkspace() { const { ydoc, provider, awareness, users } = useCollaboration({ workspaceId: 'my-workspace', user: { id: '1', name: 'John', color: '#3b82f6' } }); return ( <CollaborativeEditor ydoc={ydoc} provider={provider} awareness={awareness} user={{ name: 'John', color: '#3b82f6' }} /> ); }

✨ Features

  • Real-time synchronization: Changes sync instantly across all connected users
  • Collaborative cursors: See where everyone is editing with animated cursors
  • Presence awareness: Know who's online and what they're doing
  • Offline editing: Edit without internet, sync when reconnected
  • Conflict resolution: CRDT-based, mathematically guaranteed convergence
  • Rich text editing: Bold, italic, lists, headings, and more
  • Typing indicators: See when others are typing
  • Automatic reconnection: Handles network disruptions gracefully
